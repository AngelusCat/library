Название книги = string
ФИО автора = string
Описание = string
Год публикации = int

Название книги:
1) Проверка на пустоту: поле обязательно для заполнения;
2) Проверка на минимальное количество символов = 1
3) Проверка на максимальное количество символов = 120
4) Проверка на валидные символы: [a-zA-Zа-яёА-ЯЁ,\- :]

ФИО автора:
Минимальное значение: 5
Максимальное значение: 74
Валидные символы: [a-zA-Zа-яёА-ЯЁ \-'\(\)]

Пример: Легенда о Тиле Уленшпигеле и Ламме Гудзаке, их приключениях — забавных, отважных и достославных во Фландрии и иных странах
Пример: Путешествия в некоторые отдаленные страны света Лемюэля Гулливера, сначала хирурга, а потом капитана нескольких кораблей

Описание:
1) Может быть пустым;
2) Не обязателен для заполнения;
3) Проверять на валидные символы не надо.

Год публикации:
1) Проверка на пустоту;
2) Проверка на минимальное количество символов = 3
3) Проверка на максимальное количество символов = 4
4) Проверка на валидные символы [\-0-9]
5) Проверка на адекватность:
Год не может равняться 0;
Год может быть отрицательным числом => считается как до н.э
Год не может быть больше текущего года.

<input>:
max: Верхнее значение для ввода числа или даты;
maxlength: Максимальное количество символов разрешённых в тексте;
min: Нижнее значение для ввода числа или даты;
minlength: Минимальное количество символов разрешённых в тексте;
name: Имя поля;
pattern: Устанавливает шаблон ввода;
placeholder: Выводит подсказывающий текст;
required: Обязательное для заполнения поле;
type: Сообщает браузеру, к какому типу относится элемент формы;
value: Значение элемента.
